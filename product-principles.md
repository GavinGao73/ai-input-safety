# Product Principles

This tool is intentionally minimal and safety-oriented.

We do NOT

Store user input or output
Require login or accounts
Upload content to servers
Act as an AI intermediary
Modify original documents

We DO

Process data locally in the browser
Reduce machine-readable sensitive exposure
Generate privacy-safer representations
Discard all data after use

Security Model

When handling documents, the system prioritizes prevention of machine extraction
over file size or editability.
