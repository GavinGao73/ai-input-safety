# Privacy Overview

Filter is built around a strict local-processing model.

---

## Data Handling

- No user accounts
- No server-side processing
- No persistent storage
- No content logging

All data exists only in browser memory during the active session.

Closing or refreshing the page clears all data.

---

## Network Behavior

Filter does not transmit user content to external servers.

Any network requests are limited to static application assets required
for operation.

---

## User Responsibility

Users retain full control over their original files and generated outputs.
