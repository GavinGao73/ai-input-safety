# Privacy Model / 隐私模型 / Datenschutzmodell

---

## Core Principle  
## 核心原则  
## Grundprinzip

Filter is designed with a strict local-first privacy architecture.

Filter 采用严格的本地优先隐私架构设计。

Filter wurde mit einer konsequent lokalen Datenschutzarchitektur entwickelt.

---

## No Data Collection  
## 不收集数据  
## Keine Datenerhebung

Filter does NOT collect, transmit, or store user content.

Filter 不收集、不上传、不存储任何用户内容。

Filter sammelt, überträgt oder speichert keine Nutzerdaten.

Specifically / 具体包括 / Konkret bedeutet dies:

- No analytics on user text / 不分析文本内容 / Keine Textanalyse  
- No server uploads / 无服务器上传 / Keine Serverübertragung  
- No background requests containing content / 无内容请求 / Keine Hintergrundübertragung  
- No user profiling / 无用户画像 / Kein Nutzerprofiling

---

## Local Processing Only  
## 完全本地处理  
## Rein lokale Verarbeitung

All operations occur within the user's browser.

所有处理均在用户浏览器内完成。

Alle Verarbeitungen erfolgen ausschließlich im Browser.

Data lifecycle / 数据生命周期 / Datenlebenszyklus:

- Data exists only in memory  
- 页面刷新或关闭即清除  
- Daten existieren nur temporär im Speicher

Closing or refreshing the page clears all state.

---

## Documents & Files  
## 文档与文件  
## Dokumente & Dateien

Uploaded files are processed locally and never leave the device.

上传的文件仅在本地处理，不会离开设备。

Dateien werden lokal verarbeitet und niemals übertragen.

Raster Secure Export produces a new file without preserving recoverable text layers.

Raster Secure 导出生成的新文件不保留可提取文本层。

---

## Security vs Privacy Boundary  
## 安全与隐私边界  
## Sicherheits- und Datenschutzgrenze

Filter improves resistance to automated data extraction.

Filter 提升对自动化提取的抵抗能力。

Filter erhöht die Resistenz gegen automatisierte Extraktion.

Filter does NOT guarantee protection against:

Filter 不保证防护：

Filter garantiert keinen Schutz gegen:

- Malware on the client device  
- Compromised browsers or extensions  
- Screen capture / 摄像 / 录屏  
- Physically compromised systems

---

## User Responsibility  
## 用户责任  
## Nutzerverantwortung

Users remain responsible for verifying outputs before sharing.

用户在分享前应自行核验结果。

Nutzer sind für die Prüfung der Ergebnisse selbst verantwortlich.
