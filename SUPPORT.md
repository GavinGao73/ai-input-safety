# Support Policy / 支持策略 / Support-Richtlinie

---

## Project Nature  
## 项目性质  
## Projektcharakter

Filter is an experimental, privacy-first open-source project.

Filter 是一个实验性、隐私优先的开源项目。

Filter ist ein experimentelles Open-Source-Projekt mit Fokus auf Datenschutz.

It is NOT a commercial product and does not provide guaranteed support services.

本项目不是商业产品，不提供任何形式的支持服务承诺。

Es handelt sich NICHT um ein kommerzielles Produkt und es besteht kein Anspruch auf Support.

---

## No Service Commitment  
## 无服务承诺  
## Keine Serviceverpflichtung

The maintainers are under no obligation to provide:

维护者没有义务提供：

Die Projektverantwortlichen sind nicht verpflichtet, Folgendes bereitzustellen:

- Response to issues or questions  
- 回复问题或咨询  
- Antworten auf Anfragen oder Probleme

- Bug fixes within any timeframe  
- 在特定时间内修复问题  
- Fehlerbehebungen innerhalb bestimmter Fristen

- Feature requests or enhancements  
- 功能开发请求  
- Umsetzung von Funktionswünschen

- Compatibility with specific environments  
- 对特定环境的兼容性保证  
- Kompatibilitätsgarantien

All contributions and responses are provided on a voluntary, best-effort basis.

所有回应与改进均基于自愿与尽力原则。

Beiträge und Reaktionen erfolgen freiwillig nach dem Best-Effort-Prinzip.

## How to Seek Help  
## 如何寻求帮助  
## Hilfe & Kommunikation

If you encounter a problem, you may:

如遇问题，可以：

Bei Problemen können Sie:

1. Review existing documentation  
   查阅已有文档  
   Vorhandene Dokumentation prüfen

2. Search existing GitHub issues  
   搜索 GitHub Issues  
   Vorhandene Issues durchsuchen

3. Open a new issue with clear technical details  
   提交新的 Issue（附带清晰技术信息）  
   Neues Issue mit technischen Details erstellen

When reporting issues, include:

提交问题时建议包含：

Bei Fehlermeldungen bitte angeben:

- Environment (browser / OS)  
- 运行环境（浏览器 / 系统）  
- Umgebung (Browser / Betriebssystem)

- Steps to reproduce  
- 复现步骤  
- Reproduktionsschritte

- Expected vs actual behavior  
- 预期行为 vs 实际行为  
- Erwartetes vs tatsächliches Verhalten

## 社区协助模式  
## Community-basierte Unterstützung

Support is community-driven.

本项目采用社区协助模式。

其他用户或贡献者可能提供帮助，但不构成承诺。
